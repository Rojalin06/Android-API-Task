# Android API Integration Task - Internship Task 3

## Overview
This project is a part of my Android App Development Internship at **Maincrafts Technology**. The primary goal of this task was to integrate a REST API into an Android application using modern networking libraries and architecture.

## Features
- **REST API Integration**: Fetches dynamic post data from [JSONPlaceholder](https://jsonplaceholder.typicode.com/).
- **MVVM Architecture**: Clean separation of concerns using Model-View-ViewModel.
- **Retrofit & Gson**: Efficient network handling and JSON parsing.
- **Jetpack Compose UI**: Modern, declarative UI with Material 3 components.
- **State Management**: Handles Loading, Success, and Error states gracefully.

## Tech Stack
- **Language**: Kotlin
- **Networking**: Retrofit 2
- **JSON Parsing**: Gson
- **UI Framework**: Jetpack Compose
- **Concurrency**: Kotlin Coroutines
